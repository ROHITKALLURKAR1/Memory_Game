# Java Memory Game

A simple console-based memory game built using Java.

## How it works

1. A random 5-digit number is displayed.
2. The player has a few seconds to memorize it.
3. The number disappears.
4. The player enters the number from memory.
5. The game checks the answer and displays the result.

## Features

* Random 5-digit number generation
* Countdown timer
* User input validation
* Memory strength feedback

## Technologies Used

* Java
* Scanner
* Loops
* Conditional statements
* Thread.sleep()

## How to Run

1. Compile the program:

```
javac Memory.java
```

2. Run the program:

```
java Memory
```
